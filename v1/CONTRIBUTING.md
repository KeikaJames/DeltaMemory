# Contributing

## Commit policy

- Author must be KeikaJames only (`KeikaJames <gabira@bayagud.com>`).
- Do not add Copilot co-author trailers to commits.
- Make all changes on a feature branch and open a pull request; do not push directly to `main`.
- Run `bash scripts/install_githooks.sh` once after cloning to install the repository git hooks.
- CI must be green before a pull request can merge.
