## Design

| Check | What it tests |
|---|---|
| Gate 13A.1 (fp32) | alpha=0 patched logits == unpatched (exactly 0.0 diff) |
| Gate 13A.1 (bf16) | same, in bf16 |
| fp32 ↔ bf16 max|Δlogits| | numerical consistency across dtypes (ε ≤ 1e-2) |
| Recall@5 both dtypes | seeded facts retrievable regardless of dtype |

## Mac MPS — Deferred

Gemma-4-31B-it (~62 GB bfloat16) does not fit on Mac M-series MPS within
available unified memory budget.  The cross-machine test is replaced with
a same-GPU (spark1 CUDA) fp32 ↔ bf16 consistency check, which provides
a stronger numerical comparison (eliminates hardware micro-arch differences).
